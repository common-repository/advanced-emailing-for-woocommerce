Customize your WooCommerce emails or create new one that are sent when a condition is met.

== Description ==

https://youtu.be/nqqFXTrgcaI

With this plugin you can change the appearance of your WooCommerce emails or even create new ones that are send when a condition is met. You can edit or add new elements to the email using the drag and drop builder. No coding skills are required.


= Features =

*   **Create email templates using a drag and drop builder**: No coding skill required all is done using a drag and drop designer (similar to elementor)
*   **Custom conditions**: Define a condition for each email, it will only be sent when this condition is met. Or you can send them manually whenever you want.
*   **Style builder**: Create visually stunning emails using our style builder or save a ton of time and use one of our pre-made templates.

= Premium Features =

* **There is also a premium version that has more features, [you can get it here](https://rednao.com/downloads/advanced-emailing-for-woocommerce/)**
*   **Header Builder**: Create attention grabbing headers using images that can be translated or edited directly within the email designer
*   **Recommended Products**: Add relevant products recommendations in your emails and increase your sales.
*   **QRCodes**: Add customized qr codes
*   **Integration with other plugins**: You can use our wizard to scan orders and include specific information from other plugins into the emails.
*   **Translation ready**: Send emails in the right language even in a multilanguage site.
*   **Bulk actions**: Send emails of several orders in bulk.
*   **Access to premium template**: Instead of having to create the template yourself you can pick one from our template library.
